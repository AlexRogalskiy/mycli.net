# mycli.net

A website for mycli project.

# Running the server 

### Install the requirements file
`pip install -r requirements.txt`

### Starting the server
`./develop_server.sh start`

### Accessing
Open your favorite browser and go to http://localhost:8000.

### Stoping the server
`./develop_server.sh stop`
